# Question 1 - Linux Environment Verification

## Command 1

```bash
whoami
groups
```

### Output

```text
student
student sudo
```

### Explanation

The `whoami` command displays the logged-in username. The `groups` command shows the groups associated with the current user.

---

## Command 2

```bash
echo $SHELL
```

### Output

```text
/bin/bash
```

### Explanation

This command displays the default shell being used by the current user.
